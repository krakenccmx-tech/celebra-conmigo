# AGENTS.md

## Proyecto
Plataforma SaaS para invitaciones digitales y gestión de eventos sociales.

## Stack
- Next.js 15
- TypeScript
- Tailwind CSS
- Shadcn UI
- Supabase
- PostgreSQL
- React Hook Form
- Zod

## Reglas
- No crear código innecesariamente complejo.
- Priorizar componentes reutilizables.
- Mantener separación clara entre:
  - Public pages
  - Auth pages
  - Dashboard pages
  - Admin pages
- Todo formulario debe usar validación.
- Todo módulo debe tener estados de carga, error y vacío.
- Usar nombres claros en español para labels visibles al usuario.
- Usar nombres técnicos en inglés para archivos, funciones y variables.
- Mantener diseño minimalista, premium y responsive.

## Estilo visual
- Minimal corporate
- Inspiración Apple
- Fondos claros
- Bordes redondeados
- Sombras suaves
- Mucho espacio negativo
- Nada saturado
- Nada recargado

## Comandos esperados
- npm install
- npm run dev
- npm run lint
- npm run build

## Antes de finalizar cualquier tarea
- Verificar TypeScript
- Ejecutar lint
- Ejecutar build
- Documentar cambios
